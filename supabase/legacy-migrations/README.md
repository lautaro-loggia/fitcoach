# Legacy Supabase Migrations

These SQL files were originally kept under `supabase/migrations/` but are no longer part of the active migration chain.

They include:
- files without numeric version prefix (always skipped by Supabase CLI), and
- historical files with short versions that were normalized to full timestamped versions in `/supabase/migrations`.

They are preserved here for historical reference only.

Active migrations that the CLI applies are in `/supabase/migrations` and use numeric version prefixes.
